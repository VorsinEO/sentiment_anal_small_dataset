Этот проект был подготовлен в процессе обучения по специализации МФТИ и Яндекс по Машинному обучению и курса по изучению Flask.

Сбор данных для обучения
Сбор данных для обучения осуществлялся через сбор отзывов на мобильные телефоны с рейтингом от 1 до 5. Для сбора использовал связку request и BS4. Так же собирал с помощью Scrapy, но там выходит более громоздкая конструкция для этой задачи. Поэтому остановился именно на request+BS4. Собрал 20000 отзывов, разделил их на положительные(рейтинг 4-5)и отрицательные(рейтинг 1-3). Пропорции примерно 3 к 1, поесть минимум 5к объектов для обучения одного класса было.

Модель по анализу текста
Сначала предобработка текста с помощью pymystem3. Затем векторизация и далее Логистическая регрессия. На кроссвалидации качество сразу было выше необходимого по задаче (acc 0.85+) и остановился на этой модели. На тесте в кагале дало acc 0.97.
Дальше планирую попробовать тут применить более сложные подходы - использовать BERT через DeepPavlov или другой инструментарий. Но не хватает знаний как это делать, а копировать чужой код не доставит фана, поэтому сначала разберусь немного с этими нейронными сетями и попробую позже.

Веб-приложение
Веб-приложение или сайт подготовлен на Flask, используются WTFforms для форм ввода текста и валидация количества символов.Используется SQLAlchemy и SQLite база данных для сохранения отзывов и валидации (когда пользователь оценивает результаты работы модели). Верстка на Bootstrap.
Проект можно рассматривать как заготовку для публикации NLP моделей по работе с текстами и сборку размеченной выборки от пользователей приложения

This project was prepared in the process of training in the specialization of MIPT and Yandex "Machine learning" and Flask course from Stepik.

Text Analysis Model

The core of the project is a model pre-trained on 2000 marked product reviews from Amazon. The first part of the model is the preparation of the test using lemmatization, elimination of a number of special characters and text vectorization. The second part of the model is training several classifiers of different types and building an ensemble for text classification. More details about the model can be found in the notebook on GitHub.

Web application

The web application or website is prepared in Flask, WTF forms is used for text entry forms and character number validation. SQLAlchemy and SQLite database are used to save feedback and validation (when the user evaluates the model results). Bootstrap was used at the front.

The project can be considered as a pre-publication of NLP models for working with texts and building a marked sample from the application users.
